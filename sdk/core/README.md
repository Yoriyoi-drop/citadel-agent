# SDK Core

This directory contains the core SDK components for the Citadel Agent platform.

## Overview
The core SDK provides the foundational components for developing with Citadel Agent including:
- Base client libraries
- Authentication mechanisms
- Request/response handling
- Event system
- Core utilities
- Error handling

## Components
- `client.go` - Core client for API communication
- `auth.go` - Authentication utilities
- `request.go` - Request building and handling
- `response.go` - Response parsing and handling
- `event_system.go` - Event emission and handling
- `utils.go` - Core utility functions
- `errors.go` - Standardized error types
- `types.go` - Core type definitions

## Standards
- Consistent API interface
- Robust error handling
- Comprehensive logging
- Efficient resource usage
- Proper security measures
- Clear documentation

## Usage
The core SDK components are used by other SDK modules to provide consistent functionality.
