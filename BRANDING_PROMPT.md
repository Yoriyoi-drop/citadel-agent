## ‚úçÔ∏è Tipografi (Typography)

```
Design a typography system for Citadel-Agent platform.
Font Selection Criteria:
- Clean, modern, highly readable
- Developer-friendly (good for code and UI)
- Web-safe or good web font performance
- Professional for enterprise use

Primary Font (UI/Headers):
- Suggested: Inter, Roboto, or Poppins
- Bold weights for headers and important elements
- Clear hierarchy between different text levels

Code Font (Code display, monospace):
- Suggested: Fira Code, JetBrains Mono, or Source Code Pro
- Clear distinction between similar characters
- Good for technical documentation

Text Hierarchy:
- H1: 36px (main page titles)
- H2: 30px (section headers)
- H3: 24px (sub-sections)
- H4: 20px (component headers)
- Body: 16px (main content)
- Small: 14px (captions, labels)
- Code: 14px (monospace)
- Caption: 12px (metadata, hints)

Line height: 1.5 for body text, 1.3 for headings
Letter spacing: normal for body, slight expansion for all-caps headers
```
