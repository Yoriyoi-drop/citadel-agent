# User Guides

This directory contains user guides and tutorials for the Citadel Agent workflow engine.

## Overview
User guides provide comprehensive tutorials and how-to instructions including:
- Getting started guides
- Workflow creation tutorials
- Advanced feature guides
- Best practices documentation
- Use case examples

## Guide Categories
- `beginner/` - Getting started and basic concepts
- `intermediate/` - Advanced workflows and customization
- `advanced/` - System administration and optimization
- `tutorials/` - Step-by-step walkthroughs
- `best_practices/` - Recommended practices and approaches
- `troubleshooting/` - Common issues and solutions

## Standards
- Step-by-step instructions
- Screenshots where appropriate
- Practical, real-world examples
- Clear prerequisites
- Expected outcomes for each step

## Target Audience
- Workflow developers
- System administrators
- End users
- Integrators
- Solution architects
