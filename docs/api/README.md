# API Documentation

This directory contains API documentation for the Citadel Agent workflow engine.

## Overview
The API documentation provides comprehensive reference for all API endpoints including:
- REST API endpoints
- GraphQL schemas
- Request/response schemas
- Authentication methods
- Error codes and meanings

## API Versions
- `v1/` - Current stable API version
- `experimental/` - Experimental API endpoints
- `deprecated/` - Deprecated API endpoints

## Documentation Standards
- OpenAPI 3.0 specification format
- Comprehensive endpoint documentation
- Example requests and responses
- Authentication and authorization details
- Error code definitions
- Rate limiting information
- Version change logs

## Access Methods
API documentation is available in multiple formats:
- Interactive API explorer
- Static documentation
- OpenAPI specification files
- SDK reference guides

## Usage
Developers can use this documentation to integrate with the Citadel Agent platform programmatically.
