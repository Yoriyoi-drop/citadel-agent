# Backend Internal Structure

## Directory Purpose

This directory contains all internal packages for the Citadel Agent backend.

### Key Directories

- `ai/` - Advanced AI agent runtime and memory management
- `api/` - HTTP API handlers and controllers  
- `app/` - Main application bootstrap and initialization
- `auth/` - Authentication and authorization systems
- `config/` - Configuration management
- `database/` - Database connection and management
- `engine/` - Core workflow engine
- `repositories/` - Database repositories
- `services/` - Business logic services
- `nodes/` - Node system implementations
- `security/` - Security and sandboxing implementations
- `workflow/` - Workflow-related components
