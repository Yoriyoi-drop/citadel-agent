# Basic Database Nodes

This directory contains basic database nodes for the Citadel Agent workflow engine.

## Overview
Basic database nodes provide fundamental database operations including:
- Simple CRUD operations
- Basic query execution
- Connection management
- Basic transaction handling
- Simple data validation

## Available Nodes
- `db_query_node.go` - Executes basic database queries
- `db_insert_node.go` - Performs basic database inserts
- `db_update_node.go` - Updates records in database
- `db_connection_node.go` - Manages database connections

## Usage
These nodes are designed for basic database workflows with simple operations.

## Requirements
- Database connection credentials
- Appropriate database drivers
- Network connectivity to database

## Security & Permissions
- Requires database access permissions
- Basic connection privileges
