# Basic Logging Nodes

This directory contains basic logging nodes for the Citadel Agent workflow engine.

## Overview
Basic logging nodes provide fundamental logging operations including:
- Simple log message creation
- Basic log formatting
- Standard output logging
- Basic log filtering
- Simple log aggregation

## Available Nodes
- `simple_logger_node.go` - Creates basic log messages
- `log_formatter_node.go` - Formats log messages
- `console_logger_node.go` - Logs to console output
- `file_logger_node.go` - Logs to file output

## Usage
These nodes are designed for basic logging workflows with simple requirements.

## Requirements
- Appropriate output destinations
- Log format specifications
- Adequate storage for logs (if applicable)

## Security & Permissions
- Requires output destination permissions
- Basic logging privileges
