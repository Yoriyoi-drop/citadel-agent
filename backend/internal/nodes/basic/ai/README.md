# Basic AI Nodes

This directory contains basic AI nodes for the Citadel Agent workflow engine.

## Overview
Basic AI nodes provide fundamental artificial intelligence capabilities including:
- Simple text classification
- Basic sentiment analysis
- Simple recommendation systems
- Basic predictive models
- Text generation

## Available Nodes
- `simple_classifier_node.go` - Basic text classification node
- `sentiment_analyzer_node.go` - Simple sentiment analysis
- `basic_recommender_node.go` - Basic recommendation system
- `text_generator_node.go` - Simple text generation capabilities

## Usage
These nodes are designed for basic AI workflows with minimal configuration requirements.

## Requirements
- Basic computational resources
- Pre-trained models (if applicable)
- Input text data

## Security & Permissions
- Requires data access permissions
- Basic execution privileges
