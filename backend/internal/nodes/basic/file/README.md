# Basic File Nodes

This directory contains basic file nodes for the Citadel Agent workflow engine.

## Overview
Basic file nodes provide fundamental file operations including:
- File read/write operations
- Basic file manipulation
- File validation
- Simple file conversion
- File metadata extraction

## Available Nodes
- `file_reader_node.go` - Reads files from storage
- `file_writer_node.go` - Writes files to storage
- `file_validator_node.go` - Validates file formats
- `file_metadata_node.go` - Extracts file metadata

## Usage
These nodes are designed for basic file workflows with simple operations.

## Requirements
- Appropriate file system permissions
- Adequate storage space
- Valid file paths

## Security & Permissions
- Requires file system read/write permissions
- Basic file operation privileges
