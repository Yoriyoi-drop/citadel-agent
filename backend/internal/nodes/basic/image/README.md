# Basic Image Nodes

This directory contains basic image nodes for the Citadel Agent workflow engine.

## Overview
Basic image nodes provide fundamental image operations including:
- Image format conversion
- Basic image resizing
- Image validation
- Simple image processing
- Image metadata extraction

## Available Nodes
- `image_converter_node.go` - Converts between basic image formats
- `image_resizer_node.go` - Resizes images
- `image_validator_node.go` - Validates image formats
- `image_metadata_node.go` - Extracts image metadata

## Usage
These nodes are designed for basic image workflows with simple requirements.

## Requirements
- Valid image files
- Appropriate image processing libraries
- Adequate memory for image processing

## Security & Permissions
- Requires image file access permissions
- Basic image processing privileges
