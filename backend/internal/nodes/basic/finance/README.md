# Basic Finance Nodes

This directory contains basic finance nodes for the Citadel Agent workflow engine.

## Overview
Basic finance nodes provide fundamental financial operations including:
- Simple calculations
- Basic financial validations
- Transaction processing
- Currency conversion
- Basic reporting

## Available Nodes
- `simple_calculator_node.go` - Performs basic financial calculations
- `transaction_validator_node.go` - Validates financial transactions
- `currency_converter_node.go` - Converts between currencies
- `basic_reporter_node.go` - Generates basic financial reports

## Usage
These nodes are designed for basic financial workflows with simple requirements.

## Requirements
- Valid financial data
- Appropriate calculation parameters
- Currency exchange rate data (if needed)

## Security & Permissions
- Requires financial data access permissions
- Basic calculation privileges
