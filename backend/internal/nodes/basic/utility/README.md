# Basic Utility Nodes

This directory contains basic utility nodes for the Citadel Agent workflow engine.

## Overview
Basic utility nodes provide fundamental utility operations including:
- Simple data validation
- Basic calculations
- Format conversion
- Simple data processing
- Basic transformation operations

## Available Nodes
- `data_validator_node.go` - Performs basic data validation
- `simple_calculator_node.go` - Performs basic calculations
- `format_converter_node.go` - Converts basic formats
- `simple_processor_node.go` - Processes basic data operations

## Usage
These nodes are designed for basic utility workflows with simple requirements.

## Requirements
- Valid input data
- Basic processing parameters
- Appropriate format specifications

## Security & Permissions
- Requires data processing permissions
- Basic utility operation privileges
