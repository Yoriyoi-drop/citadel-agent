# Basic Messaging Nodes

This directory contains basic messaging nodes for the Citadel Agent workflow engine.

## Overview
Basic messaging nodes provide fundamental messaging operations including:
- Simple message sending
- Basic message queuing
- Message validation
- Simple routing
- Basic message transformation

## Available Nodes
- `message_sender_node.go` - Sends basic messages
- `simple_queue_node.go` - Implements basic message queuing
- `message_validator_node.go` - Validates message formats
- `message_router_node.go` - Routes messages to destinations

## Usage
These nodes are designed for basic messaging workflows with simple requirements.

## Requirements
- Messaging service connectivity
- Valid message formats
- Appropriate credentials

## Security & Permissions
- Requires messaging service access permissions
- Basic message sending privileges
