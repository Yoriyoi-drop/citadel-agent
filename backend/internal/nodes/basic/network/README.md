# Basic Network Nodes

This directory contains basic network nodes for the Citadel Agent workflow engine.

## Overview
Basic network nodes provide fundamental network operations including:
- Simple connectivity checks
- Basic network requests
- Host validation
- Port scanning
- Basic network diagnostics

## Available Nodes
- `connectivity_checker_node.go` - Checks network connectivity
- `simple_request_node.go` - Makes basic network requests
- `host_validator_node.go` - Validates host availability
- `port_scanner_node.go` - Scans network ports

## Usage
These nodes are designed for basic network workflows with simple requirements.

## Requirements
- Network connectivity
- Valid host addresses
- Appropriate network permissions

## Security & Permissions
- Requires network access permissions
- Basic network operation privileges
