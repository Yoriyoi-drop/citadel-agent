# Basic Text Nodes

This directory contains basic text nodes for the Citadel Agent workflow engine.

## Overview
Basic text nodes provide fundamental text operations including:
- Simple string manipulation
- Basic text formatting
- Text validation
- Simple text parsing
- Basic text transformation

## Available Nodes
- `string_manipulator_node.go` - Performs basic string operations
- `text_formatter_node.go` - Formats text content
- `text_validator_node.go` - Validates text formats
- `simple_parser_node.go` - Parses basic text formats

## Usage
These nodes are designed for basic text workflows with simple requirements.

## Requirements
- Valid text input
- Basic text processing libraries
- Appropriate formatting parameters

## Security & Permissions
- Requires text access permissions
- Basic text processing privileges
