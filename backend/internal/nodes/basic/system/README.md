# Basic System Nodes

This directory contains basic system nodes for the Citadel Agent workflow engine.

## Overview
Basic system nodes provide fundamental system operations including:
- Simple process management
- Basic system monitoring
- Environment variable access
- Simple file system operations
- Basic resource checking

## Available Nodes
- `process_manager_node.go` - Manages basic system processes
- `system_monitor_node.go` - Monitors basic system metrics
- `env_reader_node.go` - Reads environment variables
- `resource_checker_node.go` - Checks basic system resources

## Usage
These nodes are designed for basic system workflows with simple requirements.

## Requirements
- Appropriate system access
- Process management permissions
- System monitoring capabilities

## Security & Permissions
- Requires basic system access permissions
- Process management privileges
