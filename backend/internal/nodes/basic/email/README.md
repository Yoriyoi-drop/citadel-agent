# Basic Email Nodes

This directory contains basic email nodes for the Citadel Agent workflow engine.

## Overview
Basic email nodes provide fundamental email operations including:
- Simple email sending
- Basic email formatting
- Attachment handling
- Recipient management
- Basic email validation

## Available Nodes
- `email_sender_node.go` - Sends basic email messages
- `email_formatter_node.go` - Formats email content
- `attachment_handler_node.go` - Handles email attachments
- `email_validator_node.go` - Validates email addresses

## Usage
These nodes are designed for basic email workflows with simple requirements.

## Requirements
- Email service provider credentials
- Valid recipient addresses
- Network connectivity to email service

## Security & Permissions
- Requires email service access permissions
- Basic email sending privileges
