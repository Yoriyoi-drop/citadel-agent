# Basic Workflow Nodes

This directory contains basic workflow nodes for the Citadel Agent workflow engine.

## Overview
Basic workflow nodes provide fundamental workflow operations including:
- Simple workflow execution
- Basic branching
- Simple state management
- Basic task coordination
- Simple workflow analytics

## Available Nodes
- `workflow_executor_node.go` - Executes basic workflow steps
- `simple_brancher_node.go` - Implements basic branching logic
- `state_tracker_node.go` - Tracks basic workflow state
- `workflow_analyzer_node.go` - Performs basic workflow analysis

## Usage
These nodes are designed for basic workflow workflows with simple requirements.

## Requirements
- Basic workflow configurations
- Simple execution parameters
- Workflow state storage

## Security & Permissions
- Requires workflow execution permissions
- Basic state management privileges
