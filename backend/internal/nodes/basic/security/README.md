# Basic Security Nodes

This directory contains basic security nodes for the Citadel Agent workflow engine.

## Overview
Basic security nodes provide fundamental security operations including:
- Simple authentication
- Basic validation
- Credential management
- Simple encryption/decryption
- Basic security checks

## Available Nodes
- `auth_validator_node.go` - Performs basic authentication validation
- `credential_manager_node.go` - Manages basic credentials
- `simple_encryptor_node.go` - Performs basic encryption
- `security_checker_node.go` - Performs basic security checks

## Usage
These nodes are designed for basic security workflows with simple requirements.

## Requirements
- Valid credentials
- Encryption/decryption libraries
- Security check parameters

## Security & Permissions
- Requires appropriate access permissions
- Basic security operation privileges
