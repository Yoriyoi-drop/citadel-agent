# Basic Testing Nodes

This directory contains basic testing nodes for the Citadel Agent workflow engine.

## Overview
Basic testing nodes provide fundamental testing operations including:
- Simple assertion checking
- Basic response validation
- Simple test execution
- Basic test reporting
- Simple mock generation

## Available Nodes
- `assertion_checker_node.go` - Performs basic assertion checks
- `response_validator_node.go` - Validates response formats
- `simple_tester_node.go` - Executes basic tests
- `test_reporter_node.go` - Generates basic test reports

## Usage
These nodes are designed for basic testing workflows with simple requirements.

## Requirements
- Test inputs and expected outputs
- Basic validation parameters
- Test execution environment

## Security & Permissions
- Requires test execution permissions
- Basic reporting privileges
