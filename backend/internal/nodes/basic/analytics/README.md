# Basic Analytics Nodes

This directory contains basic analytics nodes for the Citadel Agent workflow engine.

## Overview
Basic analytics nodes provide fundamental data analysis capabilities including:
- Simple statistics calculation
- Basic data visualization
- Elementary trend detection
- Basic correlation analysis
- Simple forecasting

## Available Nodes
- `stats_calculator_node.go` - Calculates basic statistical measures
- `simple_chart_node.go` - Creates basic data visualizations
- `trend_detector_node.go` - Detects basic trends in data
- `correlation_analyzer_node.go` - Calculates simple correlations

## Usage
These nodes are designed for basic analytics workflows with simple data requirements.

## Requirements
- Input data in structured format
- Basic computational resources
- Statistical libraries

## Security & Permissions
- Requires data access permissions
- Basic execution privileges
