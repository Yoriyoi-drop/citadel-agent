# Basic Media Nodes

This directory contains basic media nodes for the Citadel Agent workflow engine.

## Overview
Basic media nodes provide fundamental media operations including:
- Basic format conversion
- Simple media validation
- Media information extraction
- Basic media processing
- Simple streaming operations

## Available Nodes
- `media_converter_node.go` - Converts between basic media formats
- `media_validator_node.go` - Validates media file formats
- `media_info_node.go` - Extracts media information
- `simple_streamer_node.go` - Basic media streaming

## Usage
These nodes are designed for basic media workflows with simple requirements.

## Requirements
- Valid media files
- Appropriate media processing libraries
- Adequate memory for media processing

## Security & Permissions
- Requires media file access permissions
- Basic media processing privileges
