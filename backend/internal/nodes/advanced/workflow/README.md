# Advanced Workflow Nodes

This directory contains advanced workflow nodes for the Citadel Agent workflow engine.

## Overview
Advanced workflow nodes provide sophisticated workflow operations including:
- Workflow orchestration
- State management
- Parallel execution control
- Complex routing
- Workflow analytics

## Available Nodes
- `workflow_orchestrator_node.go` - Orchestrates complex workflow patterns
- `state_manager_node.go` - Manages workflow state and persistence
- `parallel_executor_node.go` - Controls parallel execution of tasks
- `workflow_analyzer_node.go` - Analyzes workflow performance and patterns

## Usage
These nodes are designed for complex workflow management with advanced orchestration.

## Requirements
- State persistence mechanisms
- Orchestration algorithms
- Performance monitoring capabilities

## Security & Permissions
- Requires workflow management permissions
- May require access to workflow state storage
