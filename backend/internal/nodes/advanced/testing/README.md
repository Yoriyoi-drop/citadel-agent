# Advanced Testing Nodes

This directory contains advanced testing nodes for the Citadel Agent workflow engine.

## Overview
Advanced testing nodes provide sophisticated testing capabilities including:
- Load testing
- Performance testing
- Integration testing
- Chaos engineering
- Test data generation

## Available Nodes
- `load_tester_node.go` - Executes load testing scenarios
- `performance_tester_node.go` - Runs performance tests
- `integration_tester_node.go` - Performs integration testing
- `chaos_engineer_node.go` - Implements chaos engineering practices

## Usage
These nodes are designed for complex testing workflows with comprehensive test coverage.

## Requirements
- Testing frameworks and tools
- Test environment access
- Performance monitoring capabilities

## Security & Permissions
- Requires test environment access permissions
- May require performance monitoring privileges
