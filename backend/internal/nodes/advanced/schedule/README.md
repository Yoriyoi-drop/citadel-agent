# Advanced Schedule Nodes

This directory contains advanced scheduling nodes for the Citadel Agent workflow engine.

## Overview
Advanced schedule nodes provide sophisticated scheduling capabilities including:
- Complex recurrence patterns
- Resource-aware scheduling
- Priority-based execution
- Distributed scheduling
- Schedule optimization

## Available Nodes
- `advanced_scheduler_node.go` - Handles complex scheduling scenarios
- `resource_aware_scheduler_node.go` - Considers resource availability
- `priority_scheduler_node.go` - Prioritizes tasks for execution
- `distributed_scheduler_node.go` - Schedules across distributed systems

## Usage
These nodes are designed for complex scheduling workflows with advanced requirements.

## Requirements
- Sufficient system resources for scheduling calculations
- Access to resource availability information
- Time synchronization

## Security & Permissions
- Requires system scheduling permissions
- May require access to resource monitoring systems
