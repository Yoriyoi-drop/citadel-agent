# Advanced File Nodes

This directory contains advanced file processing nodes for the Citadel Agent workflow engine.

## Overview
Advanced file nodes provide sophisticated file handling capabilities including:
- File format conversion
- Batch file processing
- File validation and sanitization
- Cloud storage integration
- File transformation pipelines

## Available Nodes
- `file_converter_node.go` - Converts files between different formats
- `batch_file_processor_node.go` - Processes files in batches
- `file_validator_node.go` - Validates and sanitizes file content
- `cloud_storage_node.go` - Handles cloud storage operations

## Usage
These nodes are designed for complex file processing workflows with advanced transformation.

## Requirements
- Sufficient storage space
- Appropriate file format libraries
- Cloud storage credentials (if applicable)

## Security & Permissions
- Requires file system access permissions
- May require cloud storage access permissions
