# Advanced Utility Nodes

This directory contains advanced utility nodes for the Citadel Agent workflow engine.

## Overview
Advanced utility nodes provide sophisticated utility functions including:
- Data validation and sanitization
- Complex calculations
- Format conversion
- Data enrichment
- Content classification

## Available Nodes
- `data_validator_node.go` - Performs advanced data validation
- `calculator_node.go` - Executes complex calculations
- `format_converter_node.go` - Converts between complex formats
- `data_enricher_node.go` - Enriches data with additional information

## Usage
These nodes are designed for complex utility workflows with advanced processing requirements.

## Requirements
- Appropriate data sources
- Calculation algorithms
- Conversion libraries

## Security & Permissions
- Requires data access permissions
- May require access to external data sources
