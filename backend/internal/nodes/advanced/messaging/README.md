# Advanced Messaging Nodes

This directory contains advanced messaging nodes for the Citadel Agent workflow engine.

## Overview
Advanced messaging nodes provide sophisticated messaging capabilities including:
- Message queue management
- Real-time messaging
- Message transformation
- Protocol conversion
- Message routing

## Available Nodes
- `message_queue_manager_node.go` - Manages message queues (RabbitMQ, Kafka, etc.)
- `realtime_messenger_node.go` - Handles real-time messaging
- `message_transformer_node.go` - Transforms message formats
- `protocol_converter_node.go` - Converts between messaging protocols

## Usage
These nodes are designed for complex messaging workflows with multiple protocol support.

## Requirements
- Messaging broker connectivity
- Appropriate protocol libraries
- Network connectivity to message brokers

## Security & Permissions
- Requires messaging system access permissions
- May require network access to message brokers
