# Advanced Text Nodes

This directory contains advanced text processing nodes for the Citadel Agent workflow engine.

## Overview
Advanced text nodes provide sophisticated text processing capabilities including:
- Natural language processing
- Text analysis and sentiment detection
- Document processing
- Text transformation
- Content extraction

## Available Nodes
- `nlp_processor_node.go` - Processes text using NLP techniques
- `text_analyzer_node.go` - Analyzes text for sentiment and patterns
- `document_processor_node.go` - Processes document formats
- `text_transformer_node.go` - Transforms text content

## Usage
These nodes are designed for complex text processing workflows with AI-powered features.

## Requirements
- NLP libraries and models
- Sufficient computational resources
- Appropriate language data files

## Security & Permissions
- Requires text/document access permissions
- May require access to NLP model files
