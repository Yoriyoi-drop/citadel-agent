# Advanced Image Nodes

This directory contains advanced image processing nodes for the Citadel Agent workflow engine.

## Overview
Advanced image nodes provide sophisticated image processing capabilities including:
- Computer vision operations
- Image recognition and classification
- Image enhancement and filtering
- Format conversion
- Batch image processing

## Available Nodes
- `image_classifier_node.go` - Classifies images using AI models
- `image_enhancer_node.go` - Enhances image quality
- `object_detector_node.go` - Detects objects in images
- `batch_image_processor_node.go` - Processes images in batches

## Usage
These nodes are designed for complex image processing workflows with AI-powered features.

## Requirements
- Sufficient computational resources (GPU recommended)
- Image processing libraries
- AI model files (if applicable)

## Security & Permissions
- Requires image file access permissions
- May require access to AI model files
