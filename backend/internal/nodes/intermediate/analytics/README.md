# Intermediate Analytics Nodes

This directory contains intermediate analytics nodes for the Citadel Agent workflow engine.

## Overview
Intermediate analytics nodes provide moderately advanced data analysis capabilities including:
- Statistical analysis with configurable parameters
- Visualization with customizable charts
- Correlation analysis with multiple variables
- Forecasting with multiple models
- Advanced data profiling

## Available Nodes
- `parametric_analyzer_node.go` - Performs statistical analysis with parameters
- `custom_visualizer_node.go` - Creates customized data visualizations
- `multi_variate_analyzer_node.go` - Analyzes multiple variables and correlations
- `model_based_forecaster_node.go` - Performs forecasting using statistical models
- `advanced_profiler_node.go` - Creates detailed data profiles

## Usage
These nodes are designed for intermediate analytics workflows with configurable parameters.

## Requirements
- Structured input data
- Statistical analysis libraries
- Visualization libraries
- Computational resources for complex analysis

## Security & Permissions
- Requires data access permissions
- Statistical analysis privileges
