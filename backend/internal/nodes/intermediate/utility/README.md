# Intermediate Utility Nodes

This directory contains intermediate utility nodes for the Citadel Agent workflow engine.

## Overview
Intermediate utility nodes provide moderately advanced utility operations including:
- Advanced data validation with custom rules
- Complex calculation engines
- Data transformation pipelines
- Content enrichment services
- Data quality assessment

## Available Nodes
- `advanced_validator_node.go` - Validates data using custom rule engines
- `calculation_engine_node.go` - Executes complex multi-step calculations
- `transform_pipeline_node.go` - Processes data through transformation pipelines
- `content_enricher_node.go` - Enriches data with external information
- `quality_assessor_node.go` - Assesses data quality metrics

## Usage
These nodes are designed for intermediate utility workflows with complex processing.

## Requirements
- Custom rule definition systems
- Advanced calculation libraries
- Data transformation tools
- External data source access
- Quality metrics definitions

## Security & Permissions
- Requires data access permissions
- Calculation system privileges
