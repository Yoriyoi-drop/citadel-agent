# Intermediate Finance Nodes

This directory contains intermediate finance nodes for the Citadel Agent workflow engine.

## Overview
Intermediate finance nodes provide moderately advanced financial operations including:
- Risk modeling and assessment
- Portfolio optimization
- Financial reporting dashboards
- Compliance checking
- Transaction reconciliation

## Available Nodes
- `risk_modeler_node.go` - Builds risk models and performs assessments
- `portfolio_optimizer_node.go` - Optimizes portfolio allocations
- `financial_dashboard_node.go` - Creates financial reporting dashboards
- `compliance_checker_node.go` - Checks financial compliance
- `transaction_reconciler_node.go` - Reconciles financial transactions

## Usage
These nodes are designed for intermediate finance workflows with analytical requirements.

## Requirements
- Financial market data feeds
- Risk modeling libraries
- Portfolio optimization algorithms
- Compliance rule databases

## Security & Permissions
- Requires financial data access permissions
- Risk modeling system privileges
