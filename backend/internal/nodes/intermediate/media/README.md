# Intermediate Media Nodes

This directory contains intermediate media nodes for the Citadel Agent workflow engine.

## Overview
Intermediate media nodes provide moderately advanced media operations including:
- Video transcoding with multiple formats
- Audio processing and enhancement
- Streaming optimization
- Media metadata extraction
- Content delivery network integration

## Available Nodes
- `video_transcoder_node.go` - Transcodes videos to multiple formats
- `audio_processor_node.go` - Processes and enhances audio content
- `stream_optimizer_node.go` - Optimizes media for different streaming platforms
- `metadata_extractor_node.go` - Extracts comprehensive media metadata
- `cdn_integration_node.go` - Integrates with content delivery networks

## Usage
These nodes are designed for intermediate media workflows with advanced processing.

## Requirements
- Media processing libraries (FFmpeg, etc.)
- Streaming optimization tools
- CDN access credentials
- Computational resources for media processing

## Security & Permissions
- Requires media file access permissions
- CDN integration privileges
