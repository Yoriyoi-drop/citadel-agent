# Intermediate File Nodes

This directory contains intermediate file nodes for the Citadel Agent workflow engine.

## Overview
Intermediate file nodes provide moderately advanced file operations including:
- File format conversion between multiple formats
- Batch file processing
- File validation with complex rules
- Cloud storage integration
- File compression and archiving

## Available Nodes
- `multi_format_converter_node.go` - Converts files between multiple formats
- `batch_file_processor_node.go` - Processes large batches of files
- `complex_validator_node.go` - Validates files against complex rules
- `cloud_storage_node.go` - Integrates with multiple cloud storage providers
- `compression_engine_node.go` - Handles file compression and archiving

## Usage
These nodes are designed for intermediate file workflows with complex processing requirements.

## Requirements
- Multiple file format libraries
- Batch processing capabilities
- Cloud storage credentials
- Archiving tools

## Security & Permissions
- Requires complex file system permissions
- Cloud storage access privileges
