# Intermediate Text Nodes

This directory contains intermediate text nodes for the Citadel Agent workflow engine.

## Overview
Intermediate text nodes provide moderately advanced text processing operations including:
- Natural language processing
- Text sentiment analysis
- Document parsing and extraction
- Text classification and tagging
- Content summarization

## Available Nodes
- `nlp_analyzer_node.go` - Analyzes text using natural language processing
- `sentiment_analyzer_node.go` - Analyzes sentiment in text content
- `document_parser_node.go` - Parses and extracts information from documents
- `text_classifier_node.go` - Classifies text content by topic or category
- `content_summarizer_node.go` - Summarizes long text content

## Usage
These nodes are designed for intermediate text workflows with AI-powered text processing.

## Requirements
- Natural language processing libraries
- Sentiment analysis models
- Document parsing engines
- Classification algorithms
- Computational resources for AI processing

## Security & Permissions
- Requires text data access permissions
- NLP processing privileges
